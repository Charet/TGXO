# TGXO
一个 [teelebot](https://github.com/plutobell/teelebot) 框架的插件，用于在 [telegram](https://telegram.org/) 上管理 [hexo](https://github.com/hexojs/hexo) 博客

|  指令   |       用法       |                  介绍                  |
| :-----: | :--------------: | :------------------------------------: |
| /create | /create [文档名] |            创建一个博客文档            |
| /delete | /delete [文档名] |            删除一个博客文档            |
|  /list  |      /list       |         列出所有博客文档名         |
| /deploy |     /deploy      | 部署博客 |

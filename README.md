一个teelebot框架的插件,用于在telegram上写博客自动部署到GitHub上.

|  指令   |       用法       |                  介绍                  |
| :-----: | :--------------: | :------------------------------------: |
| /create | /create [文档名] |            创建一个博客文档            |
| /delete | /create [文档名] |            删除一个博客文档            |
|  /list  |      /list       |         输出当前所有博客文档名         |
| /deploy |     /deploy      | 将博客文档复制到HEXO中并部署到GitHub上 |
